Elirc
=====

In VERY heavy development. Pre-alpha. Risks must be explored at owns peril. Adventure awaits...

Requires Elixir 

Set the twitch chat key to the environment variables

	export TWITCH_CHAT_KEY=oauth:...

Currently runs in iex

	iex -S mix
